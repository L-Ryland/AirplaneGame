This is a program running with [bunjs](https://github.com/oven-sh/bun)

## Installing

```bash
bun install
```

To run:

```bash
bun dev
```

This project was created using `bun init` in bun v0.4.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
